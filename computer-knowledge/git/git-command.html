# 常用命令

# 查看

  • 查看用户名 git config user.name；
  • 查看邮箱 git config user.email；

# 设置

  • 设置用户名 git config --global user.name "your_username"；
  • 设置邮箱 git config --global user.email "your_email"；

# 分支

  • 查看本地分支 git branch；
  • 查看远程分支 git branch -r；
  • 查看所有分支 git branch -a；
  • 切换分支 git checkout -b 分支名称；
  • 切换本地分支 git branch -d 分支名称；
