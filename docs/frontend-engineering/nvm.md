# <img src="https://nvm.uihtm.com/images/logo.png" alt="NVM"/> NVM

一个 nodejs 版本管理工具！

1、切换 nodejs 版本

```bash
nvm use <version>
```

2、指定默认的 nodejs 版本

```bash
nvm alias default <version>
```

3、查看当前使用的 nodejs 版本

```bash
nvm current
```

4、列出已经安装的 nodejs 版本

```bash
nvm ls
```

5、列出所有可以安装的 nodejs 版本

```bash
nvm ls-remote
```

6、安装指定版本的 nodejs

```bash
nvm install <version>
```
