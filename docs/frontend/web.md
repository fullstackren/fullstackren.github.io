# Web


## webvr 框架

webVR是VR技术在浏览器的实现。

目前VR的技术实现方案仍在起草阶段，但有些技术方案几乎是铁定的。比如作为如底层的OpenGL、[WebGL][3]、[three.js][4]。

而目前就有一些基于底层技术开发出来的webVR框架，比如 [react-vr][1]和[aframe][2]。

[aframe][2]是由mozilla于15年开源的，目前已更新到0.5.0版本。

[react-vr][1]于2017年4月18日举行的facebook F8开发者大会上正式发布，并在github放出了第一个开源版本[react-vr][1]。并号称——使用react即可创造出惊人的360°景象和VR应用。

react始终是走在web巅峰的路上。
