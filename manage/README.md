# 管理

## 做好了管理能带来什么？

- **成长感**。更大的平台，能力和视野会得到提升。
- **成就感**。有团队了，能搞定更大、更复杂的事，做出更大成绩。
- **影响力**。带着团队获得更大成就，团队会更优秀，团队成员会得到成长，甚至会影响合作团队。
- **获得感**。随着能力、成绩和影响力的提升，会得到更多的物质和精神回报。

## 要不要做管理

### 外驱力

- **不得已选择**。对技术没热情，有项目经验，别人转管理，自己也要转。还有就是领导要求的。
- **别人眼里的成功**。能带团队，在别人面前很风光，别人会认为自己是个优秀的人。
- **不辜负组织**。领导说自己适合，不能辜负他和公司的期待，这是责任。
- **活轻**。事情安排下去，会轻松些。

### 内驱力

- **是否认同管理价值观**。管理需要招聘面试、辅导员工、向上汇报、开会沟通、流程梳理、资源协调、进度推动、绩效评估等管理工作，会占用大量编码时间，那你是否会觉得前面的事都属于某种意义的琐碎“杂事”，是否难以获得编码给你带来的价值感和成就感。
- **是否对管理充满热情**。是否了解过团队的工作目标、是否关心过新同事怎么培养及如何帮助他们成长、是否享受去负责一个大项目的协调和推进、是否考虑过怎样的流程或机制能应对工作中疏漏。
- **是否看中管理方面的成长**。
  - 责任。实现团队目标。
  - 视角。带好团队，关心下级、上级和平级的期待与诉求，提升看问题的系统性。
  - 思维。除了确定性思维（界限清晰、对错分明、言出必行、不出差错、靠谱），还需要对于不确定工作、模糊的执行边界、难以界定的是非对错等工作，要明确最终的目标。
