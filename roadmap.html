Web 开发

HTML

超文本标记语言（HyperText Markup Language），定义了网页内容的含义和结构。

React 生态

React

构建用户界面的 JavaScript 库。

Redux

React 状态管理

Next.js

React 零配置一站式解决方案。

Create React App

React 官方维护的创建 React 单页面应用的工具。

Preact

React 的 3kb 轻量化方案，拥有同样的 ES6 API

React-Router

React 导航组件

Nerv

Nerv 是一款基于虚拟 DOM 技术的 JavaScript（TypeScript）库，它提供了与 React 16 一致的使用方式与 API，并且拥有更高的性能表现、更小的包大小以及更好的浏览器兼容性

Blitz

基于 Next.js 构建的全栈 React 框架，灵感来源于 Ruby on Rails，具有零 API 的数据层抽象，无需 REST/GraphQL

Gatsby

基于 React 实现构建快速、现代的应用程序及网站

Redwood

将全栈引入 Jamstack，基于 React、GraphQL 和 Prisma 实现

ice.js

基于 React 的渐进式研发框架

UmiJS

插件化的企业级前端应用框架

Vue 生态

Vue3

渐进式 JavaScript 框架。

Vue2

渐进式 JavaScript 框架。

其他 Web 开发框架

Svelte

将声明性组件转换为精准高效更新 DOM 的 JavaScript 代码。

Angular

应用程序设计框架和开发平台，用于创建高效且复杂的单页应用程序。

Solid

用于构建声明式、高效且灵活 UI 界面的 JavaScript 库。。

CSS 生态

Less

Less 是 CSS 预处理语言，使 CSS 更易维护和扩展。

Sass

Sass 是 CSS 的一个扩展，它使 CSS 的使用起来更加优雅和强大。

PostCSS

通过 JavaScript 来转换 CSS，并且提供强大的插件系统。

Stylus

富有表现力、动态、健壮的 CSS。

编译构建

Rollup

新一代 JavaScript 模块打包器。

Webpack

用于现代 JavaScript 应用程序的静态模块打包工具。

turborepo

Webpack创始人下一代构建工具。

Vite

下一代前端开发与构建工具。

Gulp

基于流的自动化构建工具。

Parcel

零配置打包工具。

Babel

一个编译工具，让你可以项目中直接使用下一代 JavaScript。

Lerna

用于管理有多个包的 JavaScript 项目的工具。

计算机基础

代码检查

Standard JS

JavaScript 风格指南、检查工具和格式化工具。

StyleLint

强大的现代 CSS 检测工具，帮助样式表遵循一致的约定和避免错误。

ESLint

可组装的、用于 JavaScript 和 JSX 的代码检查工具。
