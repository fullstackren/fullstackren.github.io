# 前端UI框架选型

前端 UI 框架是一种基于 HTML、CSS、JavaScript 等前端技术的开发工具集，提供了一系列的 UI 组件、样式、布局等基础功能，使得前端开发人员可以更加高效地开发出具有良好用户体验的 Web 应用，同时也能够保证 Web 应用的视觉效果和交互体验的一致性。

1、Ant Design (opens new window)是由阿里巴巴蚂蚁金服前端团队基于 React 制作的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

2、Element Plus (opens new window) 是由饿了么团队基于 Vue.js@3 制作的 Web 应用程序 UI 库，点击查看源码 (opens new window)。

3、Ant Design Vue (opens new window)是一个基于 Ant Design 和 Vue 制作的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

4、Semi Design (opens new window)由抖音前端与 UED 团队基于 React 制作的 Web 应用程序 UI 组件库，易于定制的现代化设计系统，帮助设计师与开发者打造高质量产品，点击查看源码 (opens new window)。

5、Naive UI (opens new window)是一个基于 Vue.js@3 制作的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

6、View Design (opens new window) 是基于 Vue.js@3 制作的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

7、Layui Vue (opens new window)是一个基于 Layui 和 Vue.js@3 开发的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

8、Layui (opens new window) 是一套遵循原生态开发模式制作的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

9、Bootstrap (opens new window) 是由 Twitter 团队开发的一款基于 HTML、CSS 和 JavaScript 制作的 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

10、Vant (opens new window) 是有赞团队基于 Vue.js@3 制作的移动 Web 应用程序 UI 组件库，点击查看源码 (opens new window)。

11、Weui (opens new window) 是微信官方团队基于微信原生视觉体验而设计的一款微信应用 UI 组件库，点击查看源码 (opens new window)。

12、Taro (opens new window) 是一套遵循 React 语法规范的多端统一开发框架，点击查看源码 (opens new window)。
