# 表现展示层

技术 说明
CSS (opens new window) Cascading Style Sheets，层叠样式表，用来描述 HTML (opens new window) 或 XML（包括如 SVG (opens new window)、MathML (opens new window)、XHTML 之类的 XML (opens new window) 分支语言）中元素所呈现的视觉样式（外观和格式）以及通过外部资源实现的复杂效果。
Less (opens new window) Leaner Style Sheets，是一门向后兼容的 CSS 扩展语言，它扩展了 CSS 语言，增加了变量、Mixin、函数等特性，使 CSS 更易维护和扩展。
Sass (opens new window) Syntactically Awesome Style Sheets，是一门高于 CSS 的元语言，它能用来清晰地、结构化地描述文件样式，有着比普通 CSS 更加强大的功能。
PostCss (opens new window) 是一个使用 JavaScript 工具和插件转换样式的工具。
