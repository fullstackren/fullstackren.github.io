# pnpm

## 安装

```sh
$ npm install -g pnpm
# 或者
$ yarn global add pnpm
```

## 基本用法

### 安装、卸载和更新

- 初始化 `pnpm init`；
- 安装依赖 `pnpm install <package-name>`；
- 安装特定版本的依赖 `pnpm install <package-name>@<version>`；
- 安装所有依赖 `pnpm install`；
- 卸载依赖 `pnpm uninstall <package-name>`；
- 更新依赖 `pnpm update <package-name>`；

### 运行

- 运行脚本 `pnpm run <script-name>`；

### 查看

- 查看已安装的依赖 `pnpm list`；
